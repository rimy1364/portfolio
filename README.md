# Rimjhim Agrawal – Portfolio Website

Personal portfolio website for **Rimjhim Agrawal**, IBM Certified Maximo Solution Architect & Manager with 10+ years of EAM experience.

## 🔗 Live Site
[rimjhim-portfolio.vercel.app](https://rimjhim-portfolio.vercel.app)

## 📋 Sections
- **Hero** – Name, title, and contact info
- **About** – Summary and key stats
- **Experience** – Full career timeline (Accenture → L&T → PwC → Wipro → EY)
- **Skills** – Technical skills across 6 categories
- **Certifications** – 4 IBM certifications
- **Education** – BE in Computer Science
- **Contact** – Contact form and details

## 🛠 Built With
- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts (Inter)
- Font Awesome Icons
